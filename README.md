 Personal Expense Tracker

 Project Description
Personal Expense Tracker is a simple Python project. It helps users keep track of their daily expenses. Users can add expenses, view all expenses, check the total amount spent, and save the data in a text file.

 Features
 Add new expense
 View all expenses
 Calculate total expenses
 Save expenses in a file
 Simple menu-driven program

Technologies Used
 Python 3
 File Handling
 Functions
 Lists

 How to Run
1. Open the project folder.
2. Run the Python file:
   python expense_tracker.py
3. Select options from the menu.
